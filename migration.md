## 시스템 변경 계획

- Vue 3 + Composition API 방식으로 업데이트 + 기타 의존성들도 모두 업데이트
- 업데이트 하면서 코드 리팩토링 (작은 + 재사용성을 가지는 컴포넌트 등으로 한 개의 파일이 여러개의 책임을 가지지 않도록 함)
- 소수점 단위로 시간을 계산했지만 (1.5 = 1시간 30분) 이제는 1분 단위로 계산하도록 변경. 많이 바꿔야된다면, 사용자가 입력한 값을 6으로 나눠서 처리하는 방법도 괜찮을거같아. (만약 1시간 32분을 입력했으면, 1.53333.. 이지만 계산에는 1.5333 으로 3자리까지 적용)
- 각 입력창에 시간/분 단위로 받을 수 있도록 함 (168시간 3분 이렇게 입력할 수 있어서, 시간/분 2개의 TextField를 가지는 컴포넌트를 만들어서 관리하는 것도 나쁘지 않을거같아)
- 최종 금액 밑에 표시되는 것도 분 단위로 표시

### 가이드라인
- 반드시 이 문서 하단의 진행상황을 업데이트 할 것
- 필요한 사항/스펙이 있으면 물어보되, 커맨드 실행은 물어보지 말것
